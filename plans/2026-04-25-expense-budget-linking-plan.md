# Expense ↔ Budget Entry Linking Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a first-class link between transactions and budget entries via a M:N junction table, atomic dashboard submission via a new `/dashboard_submit` endpoint, automatic cascade soft-delete, and detail routes (`/transaction/:id`, `/budget-entry/:id`) that surface linked siblings.

**Architecture:** New `expense_budget_links` table (M:N capable, used 1:1 in Phase 1) with surrogate ID PK, no soft-delete column on the junction. Backend gains `/dashboard_submit`, `/transaction_get`, `/budget_entry_get`; existing delete handlers cascade to linked siblings; list handlers embed linked-sibling IDs per row. Frontend gets new hooks, two new pages, a `BudgetEntryCard` component, and inline expansion on the budget list.

**Tech Stack:** Cloudflare Workers, D1, Drizzle ORM, better-auth, React 18, React Query, React Router v6, Playwright, Vitest, Jest.

**Spec:** `docs/superpowers/specs/2026-04-25-expense-budget-linking-design.md`.

```ts
import { expenseBudgetLinks } from "../db/schema/schema";

async function createLinks(
  payload: SeedRequest,
  txIds: Record<string, { id: string }>,
  beIds: Record<string, { id: string }>,
  groupIds: Record<string, { id: string }>,
  payloadGroupResolver: (txAlias: string) => string,  // returns the txAlias's group alias
  db: ReturnType<typeof getDb>,
): Promise<Record<string, { id: string }>> {
  const linkIds: Record<string, { id: string }> = {};
  const now = new Date().toISOString();

  for (const link of payload.expenseBudgetLinks ?? []) {
    const id = ulid();
    const groupAlias = payloadGroupResolver(link.transaction);
    linkIds[`${link.transaction}_${link.budgetEntry}`] = { id };
    await db.insert(expenseBudgetLinks).values({
      id,
      transactionId: txIds[link.transaction].id,
      budgetEntryId: beIds[link.budgetEntry].id,
      groupId: groupIds[groupAlias].id,
      createdAt: now,
    });
  }

  return linkIds;
}
```

### Task 23: Create `TransactionDetail` page (`/transaction/:id`)

```tsx
import { useParams, useNavigate } from "react-router-dom";
import { useTransaction } from "../../hooks/useTransaction";
import { TransactionCard } from "../../components/TransactionCard";
import { useDeleteTransaction } from "../../hooks/useTransactions";

export default function TransactionDetail() {
  const { id } = useParams<{ id: string }>();
  const navigate = useNavigate();
  const { data, isLoading, isError } = useTransaction(id);
  const del = useDeleteTransaction();

  if (isLoading) return <div data-test-id="loading">Loading…</div>;
  if (isError || !data) {
    return <div data-test-id="not-found">Entry not found or you don't have access.</div>;
  }

  return (
    <div className="transaction-detail" data-test-id="transaction-detail-page">
      <button onClick={() => navigate(-1)} data-test-id="back-link">← Back</button>
      <TransactionCard
        transaction={data.transaction}
        transactionUsers={data.transactionUsers}
        linkedBudgetEntry={data.linkedBudgetEntry}
        expanded
      />
      <button
        onClick={async () => {
          await del.mutateAsync({ id: data.transaction.transaction_id });
          navigate("/transactions");
        }}
        data-test-id="delete"
      >
        Delete
      </button>
    </div>
  );
}
```
